Multivariable analyses

Models and output tables

We take each exposure significant in the univariable models at q.value (e.g., FDR adjusted p.value) < 0.05 (Smoking, Menstrual cycles changes, COVID history and menstrual cycle changes over the last year). Based on DAGS (SI5), only COVID status and menstrual cycle changes should be adjusted for confounders.
